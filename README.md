# Finance Control App

Aplicación SPA para el control de finanzas personales, desarrollada con HTML, CSS y JavaScript.

## Funcionalidades

- Gestión de categorías
- Registro de ingresos y gastos
- Balance general automático
- Persistencia de datos con localStorage
- Diseño responsive

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- localStorage

## Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio
2. Abrir el archivo `index.html` en el navegador

No requiere instalación ni servidor.

## Reglas implementadas

- Los ingresos suman y los gastos restan al balance.
- No se puede eliminar una categoría con movimientos asociados.
- Los datos se guardan en localStorage.
